# Cloud Chat

Minimal cloud chat room as a VSCode extension. Password-protected, history stored in Cloudflare D1, images auto-uploaded to a free image host.

Usage: click the ☁️ chat icon in the activity bar (left icon bar) to open the sidebar, enter the Worker URL on first use, then enter the password. You can also run `Cmd+Shift+P` → **Cloud Chat: Open**.

See the repository root README for backend deployment and full documentation.
